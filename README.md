
# Simple Web App project


This is a simple web app used to learn the basics of Django.

## Installation

1. Download the repo or clone
2. Open terminal\
2.1 py manage.py makemigrations\
2.2 py manage.py migrate\
2.3 py manage.py runserver

3. Enjoy
